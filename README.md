# Dmorrill10::Utils

Group of random Ruby mixin utility functions.

## Installation

Add this line to your application's Gemfile:

    gem 'dmorrill10-utils'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dmorrill10-utils
